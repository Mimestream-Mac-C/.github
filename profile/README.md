## Overview

Mimestream Advanced is a lightweight native macOS email client built exclusively for Gmail, providing the speed, keyboard navigation, and native integration that Gmail's web interface and generic email clients cannot match. It uses Gmail's native API directly rather than IMAP, preserving Gmail-specific features like labels, categories, send-and-archive, and threaded conversations with the performance of a native application.

Gmail API integration accesses Gmail directly through Google's API, maintaining full support for labels, categories, multiple inboxes, send-and-archive, and other Gmail features that IMAP-based clients lose or approximate poorly. The API connection provides faster sync and search than IMAP, with the real-time responsiveness that Gmail's web interface provides but in a native application.

Native macOS design provides a clean, fast interface that launches instantly, scrolls smoothly, and integrates with system features including notifications, Share menu, Quick Look, and Spotlight. The native performance eliminates the battery drain, memory consumption, and input lag that keeping Gmail open in a browser causes.

Keyboard-driven navigation provides Gmail-style keyboard shortcuts for reading, archiving, labeling, starring, and navigating messages without mouse interaction. The keyboard shortcuts match Gmail's web shortcuts for users who have internalized them, while also supporting standard macOS keyboard conventions.

For Gmail users who want native macOS performance and integration with their Gmail account while preserving every Gmail-specific feature, Mimestream Advanced is the Gmail client that treats Gmail as a first-class platform rather than a generic IMAP mailbox.

## Performance

Mimestream balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
